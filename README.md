# 文件对比备份工具

一个用于对比源文件夹和目标文件夹的 Windows 桌面工具。程序会按文件名、文件大小和修改时间找出目标端缺失或存在差异的文件，并支持勾选后复制到目标文件夹。

## 功能

- 选择源文件夹和目标文件夹
- 对比缺失文件和差异文件
- 在结果列表中勾选需要同步的文件
- 将已勾选文件复制到目标文件夹
- 自动生成对比和备份报告

## 运行源码

```powershell
python compare_backup.py
```

## 打包

```powershell
python -m PyInstaller 文件对比备份工具.spec
```
